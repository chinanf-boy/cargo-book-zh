Rust Cargo 官书（非官方翻译)

cargo-help(1)

NAME

Cargo-help - 获得 Cargo 帮助信息

SYNOPSIS

cargo help [subcommand]

DESCRIPTION

打印给定命令的帮助消息。

EXAMPLES

  1. 获取命令帮助：
    cargo help build
  1. 也可以通过--help选项：
    cargo build --help
